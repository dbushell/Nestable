Nestable

This is a forked version of DBushell's Nestable Project with important updates for handling touch-screen capable browsers that also need to function with a mouse.

Drag & drop hierarchical list with mouse and touch compatibility (jQuery plugin)

Code on GitHub

Deviations from Original

  • FIX: The mouse on touch-capable browsers would not work making the code unusable. I changed the code to use BOTH touch and mouse events simultaneously rather than the original either/or approach.
  • FEATURE: I added the capability (as have other forks) to flag a draggable item as not being allowed to have nested items below it.
  • FEATURE: I added a customActions option where buttons with custom "data-action" attributes can call methods defined during initialization (see "remove" custom action in the example below).
  • FEATURE: (by ThisIsAreku) - allow any element to be a customAction trigger with actionClass option. Also requires expand/collapse buttons to start utilizing this class.
  • Simply for presentation, I added a CDN bootstrap link on this page to use icons for expand/collapse, and I modified the CSS for gradients.
